# Project Memory

## Core
Dark Mode premium. bg zinc-950 (240 10% 3.9%) with radial indigo/violet glow.
Primary indigo-500 (239 84% 67%); accent violet-500 (258 90% 66%).
Typography: Sora 600/700 for H1/H2/H3 + display numbers (.num-display); Inter 400/500/600/700 for body, UI, buttons.
Heading rules: H1 tracking-[-0.03em] leading-[0.95]; H2/H3 tracking-[-0.02em]. Never use font-light anywhere.
Body 16-18px, font-normal/medium only. Eyebrows in Inter 12px semibold uppercase tracking-[0.18em].
Glassmorphism (bg-white/5 + backdrop-blur + border-white/10), ShimmerButton CTAs (rounded-full conic border).
Framer Motion required (FadeIn cascade, float hero image, marquee social proof).
UTM preservation via useCheckoutRedirect on every CTA — never static <a> for checkout.
